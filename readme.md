# pi provider ustc

ustc provider extension for pi coding agent. Only support deepseek v4 flash to
use

I don't want to modify the models.json manually, so I let LLM to do this slop
extension.

# Quick start

```sh
pi install git:github.com/woxQAQ/pi-ustc-provider
```

Then export your API key `USTC_API_KEY` according to shell you use
